键盘

  1. 加一个边框，然后边框外再加一个阴影。
  2. box-shadow: 1px 1px 2px 10px red;

    第一和第二个为位置，第一为阴影在本体的右边多少距离，第二位下边，皆可负数。
    第三个为模糊效果，数值越高越模糊，不需要模糊可不填写或填写0。
    第四个为阴影大本体多少像素，比如本体是个正方形，阴影大10px的话，四个边都会多出10px阴影。
    第五个为阴影颜色。

flex布局

布局的传统解决方案，基于盒状模型，依赖display属性 + position属性 + float属性。它对于那些特殊布局非常不方便，比如，垂直居中就不容易实现。
2009年，W3C提出了一种新的方案—-Flex布局，可以渐变、完整、响应式地实现各种页面布局，目前已经得到所有浏览器的支持。Flex布局将成为未来布局的首选方案。

Flex布局是什么？

Flex 是 Flexible Box的缩写，意味“弹性布局”，用来为盒状模型提供最大的灵活性。
任何一个容器都可以指定Flex布局
.box{display: flex;}

行内元素也可以使用Flex布局。
.box{display: inline-flex;}

Webkit内核的浏览器，必须加上-webkit前缀。
.box{display:-webkit-flex;display:flex;}
注意：设为Flex布局以后，子元素的float、clear和vartical-align属性将失效

用flex设置元素绝对居中(给它的父元素加下面三行代码)

.box{
display:flex;/如果是内联元素使用inline-flex
align-items: center;/水平居中
justify-content: center;/垂直居中
}

vh、vw单位

viewport height（可视高度），相对于视窗的高度：视窗高度为100vh
viewport width（可视宽度），想对于视窗的宽度：视窗宽度为100vw

background属性

background-color背景颜色。
background-position背景图像的位置。例：background-position:left top
background-size背景图片大小。例：background-size:100px 100px
background-repeat指定如何重复背景图像。例：background-repeat：repeat
background-origin指定背景图像的定位区域。例：background-origin:padding-box
background-clip背景图像的绘画区域。例：background-clip：border-box
background-attachment背景图像是否固定或者随页面的其余部分滚动。background-attachment：scroll
background-image要使用一个还是多个图像。例：background-image:url(./img/img.jpg)
这几个值在background:里不分先后

使背景图自适应

background-size:cover，使这个背景把当前元素覆盖住，例如给body元素这个背景属性，这个背景会覆盖整个body，无论浏览器大小或者显示大小多大，都会自适应覆盖住。

web窗口栏上的图标—Favicon

Favicon是favorites icon的缩写，亦被称为website icon（网页图标）、page icon（页面图标）或urlicon（URL图标）。Favicon是与某个网站或网页相关联的图标。
标准：

  1. 最初，定义一个favicon的方法是将一个名为“favicon.ico”的文件置于web服务器的根目录下，想要访问图标就去网站根目录下的Favicon.ico即可。例：浏览器url输入栏输入www.baidu.com/favicon.ico，即可访问百度的网页图标。
  2. 后来出现了一种更为灵活的方法，即使用HTML来为任何一个网页指示其图标所存储的位置，通过在页面的部分添加两个link组件来实现的。

onerror请求失败

捕抓错误信息：
img.onerror = function(xxx){console.log(xxx);xxx.target.src = 'url'}
把请求失败的img的src换成不会失败的，即自己上传的图片。

重设url以后使图标变化

首先，获取我们点击的button标签，button2 = xxx.target
然后，我们发现img标签，是button标签的兄弟标签。img2 = button2.previousSibling
然后，使变更后的url的图标覆盖已代替原来的图标。img2.src = 'http://'+ x + '/favicon.ico'

封装成函数

例：
function tag(tagName){return document.createElement(tagName)}
var div = tag('div')//就可以创建div标签了
创建一个生成标签，并给元素类名（row）的函数

divxxx = tag('div',{className:'row'})//创建一个div标签，它的classname是row,把attributes的key，也就是className赋值给element的key。

//attributes是一个属性，我们要遍历这个属性。for…in..循环可以遍历一个哈希。
element也是一个哈希，div.className='xxx',div也是个哈希，className是个key，xxx是key对应的value。
className、id、textContent都是key。
使用函数封装后，代码变成树形的，然后这样更容易理解，更容易区分，更容易排错
