数据类型（七种）

对象是复杂类型，其他的都是简单类型，复杂类型是由简单类型的组合。

  1. 数值(Number):十进制(1和.1),二进制(0b11),八进制(011),十六进制(0x11)
  2. 字符串(String):
    • 可用单或双引号。
    • 空字符串（里面什么也没有，长度为0），空格字符串（里面有个空格，长度为1）。
    • 转义符\。回车(\n)长度为1，因为只有一个字符n。如果要输出转义符(‘\‘)。
    • 多行字符串，(‘123 \ 456),456是在第二行的，第一行的\后面不能有空格等看不见的字符，否则会报错（所以不推荐使用这种方法）。也可以使用+把两个字符串连起来。（推荐使用）。
    • ES6里可以使用`（tab上面的那个键）把换行的字符包起来，里面可以直接换行，然后空格也会被算进去。
  3. 布尔值（boolean）：两个值：true和false
    • 与运算(&&)，全是真，即为真，其他为假。
    • 或运算(||)，有一个真，即为真，全是假，即为假。
  4. Symbol(符号)（es6新添加）
  5. undefined：一个值，undefined。表示为定义或不存在，即由于目前没有定义，所以此处暂时没有任何值
  6. null:一个值，null。表示空值，即此处的值为空。
    var a变量没有赋值，那么就是undefined。
    var obj = null有一个对象object，现在不给值，想以后给，就给null。(惯例，一般先给null)。

    有一个非对象，现在不给值，想以后给，就是undefined。

  7. 对象(object)：形式相当于哈希表，是一组“键值对”的集合（无序）。

对象又可以分为3个子类型。

  1. 狭义的对象(object)
  2. 数组(array)
  3. 函数(fuction)
    对象里面的valur可以是数值，字符串，布尔值，null，undefined，也可以是对象。
    key可以是空字符串，对象里的所有key都是字符串，所以加不加引号都可以，如果key是数值，那么会自动转化为字符串，如果不是数值，也不加’’的话，那么要依照标识符命名原则。

生成：var obj = {name:'hello'}。

获取：obj['name'],假设key符合标识符的条件下可以直接obj.name。

删除：delete obj['name'],删掉以后再去访问，会得到undefined。key和value都被删除了，在正常查询中obj.name输出undefined也有可能只是没有value，就像obj.name=undefined它的value就是undefined。

查询：'name' in obj , 假设得到flase，那么代表无key。

遍历key(输出顺序不确定)：
var obj = {name:'luojj',age:18} for(var key in obj){ console.log(key) }

遍历value
var obj = {name:’luojj’,age:18}
for(var key in obj){
console.log(obj[key])
}

遍历key和value

var obj = {name:'luojj',age:18} for(var key in obj){ console.log(key,obj[key])}

查看变量类型
typeof a。

null类型返回(bug1)
typeof null应该返回null类型，但返回了这是一个object类型，这是js里的一个bug（历史错误）。

函数类型返回(bug2)
typeof function(){},应该返回这是一个object类型，但返回这是一个function类型。但是function类型不在那七种类型之内。
