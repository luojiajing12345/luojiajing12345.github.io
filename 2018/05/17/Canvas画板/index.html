Posted on 2018-05-17 |

mouse事件

  • onmousedown:在用户按下了任意鼠标按键时触发。不能通过键盘触发这个事件。
    document.onmousedown = function(x){ console.log(x) }

  • onmousemove:当鼠标指针在元素内部移动时重复触发。不能通过键盘触发这个事件。
    document.onmousemove = function(x){ console.log(x) }

  • onmouseup:当用户释放鼠标按键时触发。不能通过键盘触发这个事件。
    document.onmousemove = function(x){ console.log(x) }

这些事件，前面可加on可不加on，即onmousedown和mousedown一样

画线

  1. 定义一个变量赋值为false.
  2. 按下鼠标任意键，即触发mousedown时，给变量赋值true
  3. 在鼠标指针滑动，即触发mousemove时，判断变量是否为true，即是否已按下鼠标按键，如果是，执行代码，如果不是，就不执行。
  4. 当鼠标释放时，即触发mouseup事件时，非变量赋值false，即停止鼠标指针滑动时的执行的代码。

使画板全屏

对canvas用css的宽度和高度是不适用的，并不能使canvas变为全屏画板。
var drawWidth = document.documentElement.clientWintd//获取页面宽度 var drawHeight = document.documentElement.clientHeight//获取页面高度 canvas.width = drawWidth canvas.height = drawHeight

onresize 事件

onresize 事件会在窗口或框架被调整大小时发生。
window.onresize=function(){ var drawWidth = document.documentElement.clientWintd//获取页面宽度 var drawHeight = document.documentElement.clientHeight//获取页面高度 canvas.width = drawWidth canvas.height = drawHeight }
在窗口被调整大小时，再次运行一遍获取页面宽高度的代码。

HTML DOM classList属性

classList 属性返回元素的类名，作为 DOMTokenList 对象。
该属性用于在元素中添加，移除及切换 CSS 类。
classList 属性是只读的，但你可以使用 add() 和 remove() 方法修改它。 

```document.getElementById("myDIV").classList.remove("class");```//在id为myDIV的元素中移除一个名为class的类名，可移除多个。**移除不存在的类名不会报错。**  
- Transform属性应用于元素的2D或3D转换。这个属性允许你将元素旋转，缩放，移动，倾斜等。 

### 防止移动端滑动时抖动
`body{overflow:hidden;}`
不要出现滚动条。

### 清空整个画板。
`context.clearRect(0,0,canvas.width,canvas.height)`   
使用```clerrRect()```方法清除一个和画板一样大的矩形就行了

### 把canvas保存为图片
`download.onclick = function(){
    var url = canvas.toDataURL('image/png)
    var a = document.createElement('aa')
    document.body.appendChild(a)
    a.href = url
    a.download = '图片'
    a.click()
}`

### 完全固定住，在手机上不会滑动(绝对定位住)
`#canvas{
    position: fixed;
    top: 0;
    left: 0;
}`
