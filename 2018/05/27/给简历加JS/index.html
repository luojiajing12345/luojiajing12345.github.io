关于进入页面延时

使用一个大的div盖住整个页面，当加载完页面内容时，去除这个div的classname。如果加载速度过快，可使用函数固定延时时间。

setTimeout(function(){
     firstWelcome.classList.remove('active')
 },3000)//3s后执行。

window.onscroll=function(a){console.log(window.scrollY)}

使border从左往右滑动

首先把border换成一个div（可使用::after伪元素），div绝对定位于a标签，然后个高度，宽度（和a标签的宽度一样可用100%），还有背景色，然后这个div就相当于一个border了,然后使用@keyframes给一个宽度从0到100%的过度动画就行了。

监听鼠标进入元素范围事件和走出元素范围事件

mouseenter：当鼠标光标从元素外部首次移动到元素范围之内时触发。
mouseleave：在位于元素上方的鼠标光标移动到元素范围之外是触发。
获取监听的元素，xxx.onmouseenter = function(x){let a = x.currentTarget;}。此时a为当前鼠标光标进入范围的元素。
获取它的兄弟节点：brother = a.nextSibling,有可能获取到元素与元素之间的回车,然后可以再往下找 a.nextSibling.nextSibling。此时border为a元素的兄弟元素。
获取兄弟元素标签,atag = brother.tagName,此时atag为获取到的兄弟元素的标签，但是输出是大写
判断获取到的节点是否回车,使用递归。let brother = a.nextSibling;while(brother.nodeType === 3){brother = brother.nextSibling};//3代表获取到的是文本内容，可能就是回车。
或者可以这样写let brother = a.nextSibling;while(brother.nodeType !== 'UL'){brother=brother.nextSibling};,一直找，直到找到ul标签为止，ul要大写。

解决鼠标滑到二级菜单，菜单消失问题。

从监听a标签，改为监听a标签和二级菜单标签的父标签。因为那个父标签把他们包起来了，当鼠标滑到二级菜单标签的时候，也还在哪个父标签里。
获取子元素里面的第一个ul：let brother = li.getElementsByTagName('ul')[0],
匹配选择器的所有项：let liTags = document.querySelectorAll('nav>ul>li')

建议

  1. 不要尝试第一次就把样式做出来，解决一个问题的关键是马上开始着手这个问题。一直错下去总会有对的。

使页面跳转

  1. 用锚点
    点击菜单栏中的链接时，跳转到当前页面相对应的位置。
    给a标签的href添加要跳转到的位置的标签的id。例：<a href="#siteAbout">关于</a>,<div id="siteAbout">。这样点击a标签，就会跳转到div标签开头。
  2. js
    阻止默认动作，aTag.onclick = function(x){x.preventDefault()},例如，点击a标签，要么打开新页面，要么在当前页面刷新，添加了这个后，点击a标签就什么也不发生。
    获取所需id，let a = x.currentTarget;console.log(a.href),获取到的是浏览器解析之后的地址，并不是写在a标签上的地址，这是带http协议的。要想获取用户写的用a.getAttribute('href')，这里获取的是用户写的是什么就获取的是什么。
    获取id对应的元素：let href = a.getAttribute('href');let element = document.querySelector(href)
    获取对应元素到视可的距离:let rect = element.getBoundingClientRect(),此时rect为元素的大小及其相对于视可的位置等属性，
    获取对应元素到页面头部的位置：let top = element.offsetTop
    滚动到指定位置：window.scrollTo(0,top)window.scrollTo方法用于将文档滚动到指定位置。它接受两个参数，表示滚动后的窗口左上角的坐标。
