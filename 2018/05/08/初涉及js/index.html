数组嵌套

在数组里面嵌套数组使用{},例：

var a = {
      0:{0:'a',1:'b',2:'c',lenght:3},
      1:{0:'d',1:'e',2:'f',3:'g'lenght:3}
      lenght:2
      }

数组就是一种哈希，也是一种对象。但哈希不是数组，不用有length。

  2. 获取哈希或数组里的value（值）
    数组*：index = keys[‘key’],获取全部的话，使用一个whill循环，把key换成length。 哈希**：index = keys[‘key’]，哈希的key和数组不一样，不是0,1,2,3…这样的有顺序的数字，所以正常情况去下不能遍历出来。先把哈希的key放到一个数组里当成value，然后遍历出来，再当做key来循环，两个循环相互嵌套。

防止刷新后替换的值消失。
两种情况哈希会变。一种是编辑了以后，会有新的哈希。第二种是刷新页面，新的哈希会丢失，变回原来的哈希
我们需要一编辑后，有新的哈希值就存起来，一刷新就把原来的哈希值覆盖。
localStorage.setItem(‘aaa’,JSON.stringify(hash))//后面那个是把哈希值转化为字符串,只要哈希变了，就存到aaa里面，这个localStorage里面有很多桶，这个aaa就是桶的名字。就是把哈希丢到aaa这个桶里。
在开发者工具-Application-Local Storage里面。
就是我们把变量也就是哈希存到浏览器里面，然后使用浏览器里面的覆盖掉原始的js里面的就可以了。
