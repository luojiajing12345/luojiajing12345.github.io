Posted on 2018-04-27 |

##高度是由什么决定的？

内联元素：主要是由建议行高决定，我们只能决定字体的高度，无法决定包裹字体元素的高度，这些主要由设计这些字体的设计师决定。每种不同的字体，会使包裹它的元素的高度不同
字体三部分：基线，上部，下部，还有个建议行高。每种字体的建议行高是不一样的，文字都会基线对齐。
把一个文字设置为100px，例如“hug”，从h的最上边到g的最下边为100px。
例如把span里的文字设为100px，那么span的高度会比100px高，因为会有个建议行高，每种字体的默认行高不一样。
line-height能使文字自动垂直居中。

块级元素:高度有其内部文档流元素的高度总和决定。决定，但不相等。（假设body里什么都没有，那么他的高度是0（只有个回车不算））

##文档流是什么？
文档内元素的流动方向。
内联元素从左往右流，如果流动遇到阻碍，那就换行继续从左往右流动。
如果一个元素例如，中间类截断，假设这个元素有加边框，那么那个元素会被分开，但还是一个元素，边框也还是一个，只是头在第一行，身体在第二行。
如果被截断的是英文单词，那么英文单词并不会换行，会缩到里面去。因为中文每个字是一个东西，而英文单词也是一个东西，所以默认不会被分开。如果要想使其可以被截断：

添加css属性:word-break:break-all。

块级元素从上往下流，每一块都占一行，新的一块换新一行。
把块级元素也变成从左往右流：display：inline-block（不推荐使用，建议用floot浮动）

position fixed

position属性用于指定一个元素在文档中的定位方式。
相对定位：relative。
绝对定位：absolute（基于祖先元素定位）和fixed(基于屏幕视口定位，元素位置在屏幕滚动时不会改变，会创建新的层叠上下文，脱离文档流)。
以下是基于屏幕左上角视口定位，创建新的层叠上下文

div {
position fixed;
top 0;
left 0;
}

脱离文档流，使用position fixed后，该元素会脱离文档流，脱离文档流后，其父级元素的高度就不受其影响了，因为块元素的高度为其内部文档流高度总和，脱离后就没关系了
当使用position fixd后，此元素就不会往外扩展了，它有多大，就占用多大范围。

css三角形

  1. 使一个背景图片变暗，放一个div在图片的div里，高度设置成一样，宽度不设置（它会自适应），然后把这个div的背景色设置成黑色，然后使其半透明。
    rgba()允许设定一个颜色的透明度，0=透明，1=不透明。

    rgba(255,0,0,0.1) /百分之10红色透明/
    rgba(255,0,0,0.5) /百分之50红色透明/
    rgba(255,0,0,1) /不透明，显示全红/

  2. 背景图片居中并按比例缩放
    background-position: center center第一个为水平居中，第二个是垂直居中。
    background-size: cover盖住所有面积，并按比例缩放。

  3. 宽度与最大宽度。
    假设设置一个div的宽度为1000px，width:1000px如果屏幕被缩小到小于1000px的时候，不会出现自适应，还是1000px，会出现一个滚动条，如果屏幕大于1000px的时候，它也是1000px。
    假设设置一个div的最大宽度为1000px，max-width:1000px如果屏幕被缩小到小于1000px的时候，那么它就会自适应，如果大于1000px的时候，它最大就1000px。

  4. 修改span（内联元素）宽高
    内联元素是不接受修改宽高的，要把它修改为行内块元素display:inline-block,它就接受宽高了。
    并不推荐直接设置宽高，因为如果修改里面的内容，长度不一样的话，显示会出问题，建议使用padding，这样盒子没有固定宽度的话，增加了内容宽度也会增加，实现自适应。但为了防止不同浏览器出现不一样的高，我们可以使用line-height固定一下高度

  5. 画三角形

    写个div，然后里面没有内容，就是height：0px，width: 0px。
    然后设置一个大边框border: 10px solid。
    然后给每个边框不一样的颜色border-top-color:bule，就出现4个等腰三角形了。
    然后把三个设置为透明就可以了。

    还可以设置另一种三角形，就是把一个边框宽度大小设置为0pxborder-top-width:0px，然后再把两个设置为透明。

我们在span里面最好不要放div,容易出bug。span加个display:block就是div了。

  1. 绝对定位

    当一个元素在另一个元素里面，我们需要把子元素拿出来，并依附着父元素定位是，就要用到绝对定位（脱离文档流，然后定位）。

    在子元素上面写position: absolute。
    在父元素上面写position: relative。
    这就是子元素相对于父元素定位。

    如果要紧贴父元素下方，子元素上写top:100%,紧贴左边right:100%,依此类推。。。

    默认情况下div就有个::before和::after，但是它是隐藏的。

  2. box-sizing
    box-sizing 属性用于更改用于计算元素宽度和高度的默认的 CSS 盒子模型。

    box-sizing: content-box

默认值，标准盒子模型。width与height只包括内容的宽和高，不包括边框（border），内边距（padding），外边距（margin）。

box-sizing: border-box

width和height属性包括内容，内边距和边框，但不包括外边距。
