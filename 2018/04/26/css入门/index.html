引进CSS的三/四种方式

  1. 内联样式
    \<h1 style="color:red"\>hello world\</h1\>，在开始标签处添加样式。

  2. 内部样式表。
    \<head>\<style\>p \{color:red;\}\</style>\</head\>,在head标签里加入一个样式表style标签。

  3. 外部样式表
    \<head>\<link rel="stylesheet" type="text/css" href="mystyle.css">\</head>,在head标签里加一个link标签，新建一个css文件，链接到这个文件。

4.在css文件里链接到另一个css文件

@import url(./b.css)
链接到当前同目录下的b.css文件

关于选择器

\<h1 class="key" id="principal">Hello World\</h1>

  1. 标签选择器

    h1 {
    color:red;
    }

  2. 类选择器
    给元素设置class属性，可以为元素指定类名，文档中多个元素可以拥有同个类名

    .key {
    color:red;
    }

  3. ID选择器
    通过设置元素的ID属性为该元素指定ID，每个ID在文档中都是唯一的
    #principal {
    color:red;
    }

关于优先级：

如果多于一个规则指定了相同的属性都应用到一个元素上，css规定拥有更高确定度的选择器优先级更高。

ID选择器>类选择器>标签选择器

关于选择子标签

类选择器：类名与类名之间隔一个空格就可以了，只能递级下降，例：

.爷爷 .爸爸 .我

标签选择器：使用“>”连接，也是递级下降。例：

爷爷>爸爸>我

他们之间可以混合使用，例：

.爷爷 .爸爸>我

常见制作菜单栏，使列表浮动后

  1. 清除列表小圆点
    list-style:none
  2. 查看bug，给列表加边框
    border:1px solid red;

  3. 去除浮动后bug的方法（添加伪类），clearfix为浮动列表的父类元素（ul）的类名.(就是给浮动元素的爸爸添加以下样式)

    .clearfix::after{
    content: ‘’;
    display:block
    clear:both
    }

  4. 浏览器上查看字体的大小。

    打开开发者工具
    点击要查看大小的字体的元素
    点击右边栏的Computed（展示计算它的样式，style为展示所有的样式）
    在Filter一栏中输入font-size，看到的就是了

  5. 去掉链接下划线。

    text-decoration:none(用于设置元素中的文本的修饰类型)。

  6. 使菜单栏每个选项在鼠标移到上面的时候有下划线。

  • 先给每个链接加个边框,并设置透明色。（如果不先加边框的话，到下一步的时候鼠标移上去菜单栏会动，因为相当于无中生有忽然出现个边框，会把旁边的选项挤开）。

    border:1px solid transparent，使边框透明。

  • 只设置一个下边框透明就可以了。
  • 添加一个:hover伪类，这个伪类的作用是，当鼠标停在使用这个伪类的元素上时，会出现的效果。
  • 添加下边框颜色和宽度就可以了。
  1. 修改下划线与链接高度。
    在网页中只有鼠标放在链接文本上才会出现下划线，那么怎么测试高度呢？
    打开开发者工具———找到并点击代码中的链接文本————点击右边的:hov————然后勾选:hover。那个链接文本就一直处于鼠标指向它的状态了。
    然后使用工具测出距离，再把距离拉开，然后发现li标签没有把a标签包裹着，然后使用display：block给a标签，让他变成一个块元素就好了。

color: inherit;是继承的意思，就是祖先什么颜色，他就什么颜色。
例如：a标签本身是有浏览器给它的默认属性的，所以他不会继承他祖先的属性，但是我们让他要继承属性，因为我们的权限最大，所以他就继承了祖先的。（有些属性可以继承，有些属性不可以继承）
