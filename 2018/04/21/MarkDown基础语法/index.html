MarkDown基础语法

Posted on 2018-04-21

title: Markdown基本语法

date: 2018-04-21 10:59:19

MarkDown定义：

  • MarkDown是一种轻量级语言。
  • 它使我们专心于码字，用[标记]语法，来代替常见的排版方法，可以使普通文本内容具有一定的格式。

为什么要学习MarkDown（优点）？

  • 纯文本内容，兼容所有的文本编辑器与字处理软件
  • 可以放在版本管理系统中，追踪历史变更
  • 轻松的导出HTML、PDF和本身的.md文件
  • 简洁、高效、可读、直观、学习成本低

Markdown语法主要分为如下几大部分：

1. 标题

#对应h1

##对应h2

###对应h3

####对应h4

#####对应h5

######对应h6
符号与文字之间要加一个空格

2. 段落

段落前后都要有空行，若想在段内强制换行，使用两个或以上空格加回车

3. 区块引用

在文字每行或只在第一行使用符号“>”,还可以使用多个嵌套使用“>>”,“>>>”等；符号与文字之间要用一个空格隔开。

4. 代码区块

代码区块要每行加上四个空格或一个制表符（tab）。如：
function print(){
console.log(“Hello World”);
}

5. 强调

加粗使用4个或_把文字包起来。
斜体使用2个或_

6. 列表分割线

使用三个或以上的*，或-或_。

7. 链接

链接有两种形式：行内式和参考式。
行内式

[name](url),例：
[这是百度](https://www.baidu.com)

参考式

[name](key)
[key]:url
例：
[这是百度](a)
[a]:https://www.baidu.com

8. 图片

添加图片的形式与链接相似，只需在链接的基础上前方加一个“！”。
9. 反斜杠\
相当于反转义作用，使符号变成普通符号。

10.列表

有序列表使用1.和2.和3.。
无序列表使用-或*或+加个空格
