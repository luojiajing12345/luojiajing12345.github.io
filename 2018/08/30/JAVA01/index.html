java学习路径

发表于 2018-08-30 | 分类于 Java

第一阶段：模仿和使用(学习不懂的东西，第一件事就是模仿)

  1. 学习java基本语法和特性（不懂本质，但是会用那些东西）

    • 学习简单的算法和数据结构，用编程网站简单题做训练
    • 尽量只关理解java中的一些概念
    • 勤于动手，多做数据结构和算法训练
    • 遇到困难，多思考

  2. 学习面向对象编程（OOP）

    • java是一门强面向对象的语言，所有东西都写在一个类中
    • 记忆常见的设计方法
    • 尝试去理解其设计理念

  3. 学习一个框架的使用(要求：会用)

    • MVC设计模式（会）
    • 网络编程（会）
    • 使用数据库（会增删改查）

  4. 推荐书籍

    • 《java核心技术》
    • 《Head First Java》

第二阶段：思考并理解（为什么？）

  1. 深入理解java语言的特性

    • 《java编程思想》（推荐）
    • 《Effective java》

  2. 有体系的学习数据结构和算法（思想）

    • 理解数据结构和算法的应用场景
    • 刷题训练
    • 模板化（不要十个一样的场景用十个不同的算法，容易出错）

  3. 学习一些设计模式

    • 尝试理解其设计理念
    • 《Head first Design》
    • 《大话设计模式》

  4. 学习操作系统相关知识

    • 理解操作系统底层的设计逻辑
    • 提高程序设计质量
    • 《深入理解计算机系统》
    • 《深入理解java虚拟机》
    • 《计算机网络》
    • 《数据库系统概念》

第三阶段:领悟加设计

  1. 清楚程序执行过程中看不见的东西

    • 代码的编译和执行过程
    • 系统底层的内存分配和调度
    • 阅读JDK源码及使用的框架的源代码

  2. 可以根据不同的任务选择最好的工具

    • 编程语言
    • 框架等

  3. 根据自己的兴趣向外拓展

    • 自己设计一些框架
    • 学习分布式系统
    • 学习函数式编程等

总结

  1. 招式和内功兼修
  2. 高效不意味快速，有长久的学习计划
  3. 学习过程中善于总结思考
